# 🔍 UX Audit Case Study — Pinterest & Amazon

This case study presents a usability audit of two high-traffic digital platforms — Pinterest and Amazon — to identify UX friction points and propose improvements focused on usability, clarity, and user flow efficiency.

---

# 🎯 Objective

Evaluate real-world UX issues in discovery and transaction-heavy platforms through heuristic analysis and observational audit.

Focus Areas:
- Usability friction points  
- Information hierarchy  
- Cognitive load reduction  
- User journey optimization  

---

# 🧠 Audit Method

- Heuristic Evaluation  
- Interface Observation  
- User Journey Analysis  
- Problem Identification → Improvement Recommendations

---

# 📌 Platform 1 — Pinterest UX Audit

## Problem Identified

### 1. Content Overload
The home feed presents excessive visual information, increasing cognitive load and reducing scanability.

### 2. Weak Visual Hierarchy
Content prioritization is unclear, making discovery less efficient.

### 3. Search Accessibility
Search functionality could be made more prominent to improve navigation and content discovery.

---

## UX Improvements Proposed

- Improve feed structure for better content scanning  
- Strengthen visual hierarchy through spacing and prioritization  
- Optimize homepage layout for discoverability  
- Redesign navigation to reduce cognitive effort  

---

## Expected Impact
- Better content discoverability  
- Reduced information overload  
- Improved engagement and navigation clarity

---

# 🛒 Platform 2 — Amazon UX Audit

## Problem Identified

### 1. Checkout Friction
Multi-step checkout introduces unnecessary complexity.

### 2. Error Recovery Gaps
Payment and checkout errors could provide clearer guidance.

### 3. Information Overload
Product and checkout screens contain competing information that increases decision fatigue.

### 4. Flow Disruptions
Navigation and checkout progression can create uncertainty for users.

---

## UX Improvements Proposed

- Simplify checkout flow structure  
- Improve error handling and recovery  
- Reduce distraction during purchase flow  
- Enhance clarity in pricing and order details  

---

## Expected Impact
- Reduced checkout friction  
- Improved conversion flow  
- Better trust and usability during purchase

---

# 🛠️ UX Principles Applied

- Usability Heuristics  
- Information Architecture  
- Visual Hierarchy  
- Cognitive Load Reduction  
- User Flow Optimization  

---

# 📈 Key Learning

This audit reinforced how small usability issues in high-traffic products can significantly impact user behavior, engagement, and conversion outcomes.
